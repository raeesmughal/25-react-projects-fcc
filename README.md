# 25 React Projects Challenge 🚀

This repository is a comprehensive collection of **25 React projects** built as part of a daily practice routine to master React fundamentals, hooks, and advanced UI/UX patterns.

## 🛠️ Tech Stack
* **Core:** React.js (Vite)
* **Styling:** Pure CSS
* **Backend (Final Project):** MongoDB, Express.js, Node.js

## ⚙️ Development Workflow

I follow a professional Git workflow for every project in this repository:
1.  **Branching:** Each project is built in a dedicated feature branch.
2.  **Commits:** Atomic commits with descriptive messages.
3.  **Pull Requests:** Code is reviewed and merged to the `main` branch via PRs to simulate a real-world team environment.
